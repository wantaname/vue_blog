## vue个人博客写作平台

### why

+ 想打造自己的博客平台
+ markdown格式比富文本扩展性更强
+ 动态博客框架wordpress不好用
+ Vuepress、Hexo等静态博客框架，每次发布都需要编译
+ 相比展示，更需要一个舒适的写作环境
